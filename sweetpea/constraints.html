Constraints¶

at_most_k_in_a_row(k, levels)¶

Constrains an experiment to allow at most k consecutive trials with the same level among those in levels.

Parameters

  • k (int) – the maximum number of consecutive repetitions to allow

  • levels (Union[Factor, Tuple(Factor, Any)]) – either a factor or a tuple of a factor and a level name within the factor; specifying just a factor is the same as listing all of the factor’s levels as separate constraints

Return type

Constraint

no_more_than_k_in_a_row(k, levels)¶

The same as at_most_than_k_in_a_row().

exactly_k_in_a_row(k, levels)¶

Constrains an experiment to allow levels among those specified in levels only when they appear as part of a sequence of exactly k consecutive trials with the same level.

Parameters

  • k (int) – the number of repetitions to require

  • levels (Union[Factor, Tuple(Factor, Any)]) – either a factor or a tuple of a factor and level names within the factor; specifying just a factor is the same as listing all of the factor’s levels as separate constraints

Return type

Constraint

exactly_k(k, levels)¶

Constrains an experiment to allow levels among those specified in levels only when they appear as part of a sequence of exactly k trials with the same level in the experiment.

Parameters

  • k (int) – the number of repetitions to require

  • levels (Union[Factor, Tuple(Factor, Any)]) – either a factor or a tuple of a factor and level names within the factor; specifying just a factor is the same as listing all of the factor’s levels as separate constraints

Return type

Constraint

exclude(factor, level)¶

Constrains an experiment to disallow the specified level.

Parameters

  • factor (Factor) – the factor whose levels are named by levels

  • levels – a level name among the levels in factor

Return type

Constraint
